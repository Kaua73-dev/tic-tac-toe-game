# Overview

### This is a simple Tic Tac Toe game implemented in Java.
#### The project was created to practice core Java concepts such as object-oriented programming, records, validation, loops, and basic I/O handling.

### The game runs in the console and allows two players to play alternately until one wins or the match ends in a draw.

# Features

- 3x3 board

- Two-player mode (X and O)

- Input validation for coordinates

 ## Detection of:

- Horizontal win

- Vertical win

- Diagonal win

## Draw

### Clean board rendering in console

### Basic error handling for invalid moves
